# TeslaNetEaseMusic

在特斯拉车机上听网易云音乐。深空霓虹科技风、行车滚动歌词,登录你自己的账号。

**功能**
- 日推 / 我的歌单 / 搜索 / 私人FM(一首接一首的个性化电台)
- 播放 / 暂停 / 上一首 / 下一首 / 乱序 / 循环 / 进度 / 音量
- 卡拉OK式滚动歌词,纯音乐会显示“纯音乐 · 请欣赏”
- 大按钮、深色高对比,方便车上点;记住上次听的歌和音量

## 怎么跑起来

按你手头的设备选一种,详细步骤都在 [DEPLOY.md](DEPLOY.md)。

| 你的情况 | 用哪种 | 特点 |
|---|---|---|
| 有一台 **Mac / 电脑**,先试试 | [Mac 一键脚本](DEPLOY.md#mac) | 最快,几分钟能在车上打开;用车时电脑要开着 |
| 有 **群晖 NAS** | [群晖部署](DEPLOY.md#nas) | 常年在线、地址固定;**不需要公网 IP** |
| 有 **VPS / 云服务器** | [VPS 部署](DEPLOY.md#vps) | 稳定,地址固定 |

最快体验(在有 Docker 的 Mac 上):

```bash
git clone https://github.com/JovanCai/TeslaNetEaseMusic.git
cd TeslaNetEaseMusic
./deploy.sh
```

脚本会启动服务、让你用手机扫码登录、然后打印一行 `https://…` 地址。把这行地址在**特斯拉车机浏览器**里打开就能听。

## 常见问题

- **要重新登录 / 换账号**:运行 `./relogin.sh`,再扫码。
- **更新到新版本**:`git pull` 后重新 `./deploy.sh`(登录状态保留)。
- **80 端口被占用**:在 `.env` 里设 `APP_PORT=8080`(或别的空闲端口)。
- **海外听灰色歌**(两种,默认都关,自己决定开不开;改完重新部署):
  - 有版权、海外变灰的歌:`.env` 设 `REGION_UNLOCK=true`(区域解锁,后端自动用随机中国 IP,靠你自己账号版权)。
  - 无版权的灰歌:`.env` 设 `ENABLE_UNBLOCK=true`(从 QQ/酷狗等找同名歌替换)。匹配可能串版本、音质不一。

## 关于隐私和版权

- 每个人部署自己的一份,登录自己的网易云账号。
- 登录 cookie 只保存在你自己服务器的数据卷里,车机浏览器里没有它。
- 播放依赖你自己账号的版权;区域解锁默认关闭,开不开由你决定。

设计与实现文档见 `docs/`。
